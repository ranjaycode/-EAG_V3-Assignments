# Aura AI: Gemini 2.5 Flash Chrome Extension

Aura AI is a high-performance, premium Chrome extension that leverages the speed and intelligence of Google's Gemini 2.5 Flash to enhance the browsing experience. 

## 🌟 Key Features
- **Pulse Summary**: Instant high-level summary of any long-form article or web page using the cutting-edge **Gemini 2.5 Flash** model.
- **Contextual Side Panel**: A modern, side-docked interface that allows you to interact with AI without leaving your current tab.
- **Deep Research Chat**: Ask specific questions about the content you're viewing for immediate, context-aware answers.
- **Premium Aesthetics**: A custom-designed, glassmorphic UI with neon violet accents and smooth animations.

## 🚀 Installation Guide
1. Open Google Chrome and go to `chrome://extensions/`.
2. Enable **Developer mode** (top right corner).
3. Click **Load unpacked**.
4. Select this project folder: `c:\Users\dell\Desktop\EAGV3\Session2Assignment`.
5. Pin **Aura AI** to your toolbar for easy access!

## 🛠️ Technical Architecture
- **AI Model**: `gemini-2.5-flash` (via Google Generative Language API v1beta).
- **Extension Framework**: Manifest V3 + Side Panel API.
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), and ES6+ JavaScript.
- **Content Engine**: Targeted DOM traversal for efficient context extraction.
